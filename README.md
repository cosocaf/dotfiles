# dotfiles

## Overview

My dotfiles.

## Usage

### Windows

Run the following in PowerShell.

1. Install git

```pwsh
winget install -e --id Git.Git
```

Set up your own git configuration.

2. Download

```pwsh
cd /path/to/your/dev-dir
git clone https://github.com/cosocaf/dotfiles.git
cd dotfiles
```

3. Setup

```pwsh
./win-setup.ps1
```
